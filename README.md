## Background

The "intention economy" (Simon Taylor) is an emerging commerce paradigm where AI agents — not humans — are the buyers. They arrive with intent already formed, a budget, and constraints. The merchant's only job is to fulfill it.

Recent data (March 2026):
- x402: 75M transactions, $24M volume, 22K sellers (30 days)
- MPP (Stripe + Tempo): 913 agents, 34K transactions in week 1
- AliPay: $120M/week in AI-enabled transactions
- McKinsey: Trillion-dollar opportunity

## Key Protocols

| Protocol | Purpose |
|---|---|
| x402 (Coinbase) | On-chain payment settlement |
| MPP (Stripe + Tempo) | Machine Payments Protocol — crypto + fiat |
| A2A (Google, Anthropic, OpenAI) | Agent-to-agent communication |
| AP2 (Google) | Intent mandates + audit trail |
| ERC-804 | Agent registry/discovery |
| Agent Pay (Mastercard) | Card token + agent authorization |
